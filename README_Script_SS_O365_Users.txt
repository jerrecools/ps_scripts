Vooraleer je het script uitvoert:

1.	Download en installeer Microsoft Online Services Sign-In Assistant for IT Professionals RTW:
	a.	https://www.microsoft.com/nl-nl/download/details.aspx?id=41950
2.	Installeer de Microsoft Azure Active Directory Module for Windows PowerShell:
	a.	Open PowerShell als administrator
	b.	Voor commando   Install-Module MSOnline   uit.
3.	Lees in Smartschool de helppagina over “Webservices”. Je moet mogelijk nog een aantal zaken configureren.
4.	Users in SS en O365 moeten dezelfde opbouw hebben qua gebruikersnaam.
	a.	Bv    voornaam.achternaam

In de template staan een aantal zaken die je moet aanpassen voor je eigen school. Deze worden voorafgegaan door twee *. 
Je zoekt gewoon naar de sterretjes (ctrl + F), verwijdert deze en vult je schooleigen gegevens in.
	Bijvoorbeeld: 
		**$gebruikersnaam@domein.be   ==>   $gebruikersnaam@vibo.be
